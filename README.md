# Seminário — Redes de Computadores

**Tema:** Enhancing Network Security: A Stacked Ensemble Intrusion Detection System for Effective Threat Mitigation

## Descrição

Este repositório reúne os materiais do seminário apresentado na disciplina de Redes de Computadores. O trabalho aborda sistemas de detecção de intrusão (IDS) baseados em aprendizado de máquina, especificamente o uso de **stacked ensemble learning** para identificar e mitigar ameaças em redes de computadores de forma mais eficaz que abordagens tradicionais.

O artigo discute como a combinação de múltiplos classificadores em uma arquitetura de empilhamento (*stacking*) aumenta a precisão na detecção de ataques, reduzindo falsos positivos e melhorando a capacidade de resposta a ameaças desconhecidas.
